## Som van de derde machten

Schrijf een programma dat de som van de eerste $$n$$ derde machten berekent en print.

Voor $$n=4$$ geeft dit:
	$$1^3 + 2^3 + 3^3 + 4^3$$
 


### Voorbeeld1

#### invoer

```console?lang=python&prompt=>>>
4
```

#### uitvoer

```console?lang=python&prompt=>>>
100
```

### Voorbeeld2

#### invoer

```console?lang=python&prompt=>>>
24
```

#### uitvoer

```console?lang=python&prompt=>>>
90000
```
